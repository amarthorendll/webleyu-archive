# webleyu-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面并非针对任何单个域名或具体网站，而是作为通用存档库，存放各类静态 HTML 资源。

## 项目简介

webleyu-archive 是一个静态 HTML 页面归档仓库。主要目的是集中存储和展示独立的 HTML 文件，便于版本管理、历史回溯和内容分发。仓库本身不涉及后端服务或动态内容，所有页面均为纯前端静态文件。

## 目录说明

```
webleyu-archive/
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个独立 HTML 页面可置于该目录下，建议按类别或时间进行子目录组织。
- `assets/`：存放页面共享的静态资源，避免重复引用。
- `index.html`：可选的仓库首页，用于展示页面列表或导航。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，不依赖外部服务或特定域名。
- 页面内容可能包含纯文本、图表、交互组件等，但不会收集用户数据或进行追踪。
- 归档的页面可能来自历史备份、测试样例或临时展示需求，不保证其内容持续更新或准确性。
- 页面可能存在设计或功能上的不完善，仅供浏览或参考。
